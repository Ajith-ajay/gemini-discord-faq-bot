# Frequently Asked Questions

## What is this server for?
This is a server for fans of the Gemini API and for testing this bot.

## How do I use the bot?
You can interact with me using slash commands. Try `/faq` to ask a question, or `/ping` to check my status.

## What is the Gemini API?
The Gemini API is a new API from Google that allows you to interact with the Gemini family of models.

## What technologies was this bot built with?
This bot was built with TypeScript, Node.js, Discord.js, the Google Gemini API, and it is fully containerized with Docker.

## Can the bot remember previous questions?
Yes! I can remember the context of our current conversation. You can ask me follow-up questions, and I'll understand what you're referring to.

## How do I clear my conversation history?
You can use the `/clear` command to erase your conversation history with me and start fresh.
